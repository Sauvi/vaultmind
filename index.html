Features
Everything you need.
Nothing in the cloud.
Chat with any document
Ask questions in plain English. Works with PDF, DOCX, TXT, and Markdown files. Get precise answers, not hallucinations.
Legal contract review
Automatically extracts parties, dates, risk clauses, obligations, and governing law. Two-stage: instant pattern scan + AI analysis.
Streaming responses
Answers appear word by word — just like ChatGPT. No staring at a spinner. Feels fast even on modest hardware.
Permission-first design
Every action shows a dry-run plan before executing. You see exactly what will be read and written. Then you approve.
Local audit log
Every action is recorded locally — what file, what action, when, success or failure. Full transparency, no external logging.
Behavior memory
VaultMind learns which actions you use most and suggests them automatically. Gets smarter about your habits over time.
Privacy architecture
Your data never
leaves this box.

Every step of the pipeline runs on your local machine. There is no API call to OpenAI, no upload to S3, no telemetry. Not even to us.

Get started
Up and running
in 4 steps.
01
Install Ollama
Download from ollama.com. Then pull the recommended AI model.
ollama pull phi3.5
02
Clone VaultMind
Clone the repo and install Python dependencies.
pip install -r requirements.txt
03
Launch
Run the start script. VaultMind opens in your browser automatically.
run.bat (Windows)
bash run.sh (Mac/Linux)
04
Drop a file
Drag any document onto the interface. Approve the action. Get answers.
